Главная страница

Данное приложение служит для реализаций функционала при работе с БД "Кинотеатр".

Приложение расчитано для того чтобы им в первую очередь использовали сотрудники кинотеатра.
Для клиентов необходимо ограничить функционал.
Благодаря пользователи данного приложения способны:
1. Просматривать список всех фильмов, которые сейчас находятся в базе.
2. Добавлять новый фильм в базу данных.
3. Просматривать список доступных сеансов.
4. Редактировать текущие сеансы.
5. Оформлять заказы.
6. Просматривать статистику и загружать отчёты.

C помощью данного приложения возможно автоматизировать работу отдельного кинотеатра.
